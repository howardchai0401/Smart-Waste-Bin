# Smart Waste Bin ESP32 Firmware

## Features

- ESP32 DevKit
- HX711 + 4 Load Cells
- HC-SR04 Ultrasonic Sensor
- MQ-135 Gas Sensor
- WiFi Support
- MQTT Ready
- JSON Data Output

## Hardware

- ESP32 DevKitC
- HX711
- 4 × 50 kg Load Cells
- HC-SR04
- MQ-135
